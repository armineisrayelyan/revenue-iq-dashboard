# RevenueIQ

RevenueIQ is a premium SaaS analytics dashboard designed for subscription-based businesses.

## Users

- Admin
- Manager
- Viewer

## Features

- Dashboard
- Customers
- Revenue
- Subscriptions
- Reports
- Settings

## Goals

Help SaaS companies monitor:

- Monthly recurring revenue
- Annual recurring revenue
- Customer growth
- Churn rate
- Conversion rate
- Active subscriptions
- Recent payments
